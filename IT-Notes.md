# Multi-App Launcher Script (.bat)

This repository contains a simple Windows Batch script to automate opening multiple programs with a single click.

## The Script
To create this, copied the code below into my **Notepad**:

# batch
@echo off
:: Step 1: Prevent commands from cluttering the terminal

@echo off

step 2: List the programs you want to open

- start putty
- start vlc
- start notepad

Note: You can add any program name or file path here
